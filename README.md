## Docs
Contiene las capturas de pantalla de las pruebas unitarias asi como la ejecución del programa y la de otro grupo

## Main.java
Es el programa que sirve como tablero para probar las funciones de la Clase que im.plemente al interfaz de radio, aquí es donde se sustituye la Clase que crearon los otros grupos para realizar las pruebas

## Radio.java
Es la interfaz que contiene la definición de las firmas necesarias para crea la Clase Radio

## RadioImp1.java
Es la clase que creamos para implementar la interfaz Radio, contiene la lógica necesaria para el funcionamiento de Radio, los javadocs y es la clase que se utilizó para crear las pruebas unitarias

## RadioImp1Test.java
Es la clase que realiza las pruebas unitarias de RadioImp1.java

## RadioImp2.java
Es la implementación proporcionada por otro grupo y que implementa la misma interfaz Radio.java
